# Assets Folder

Organize your game assets into the subfolders below:

## Images/ folder:
Place all PNG/AVIF image files here:
- **player.png** - Spaceship 1 (Character selection option 1)
- **player2.PNG** - Spaceship 2 (Character selection option 2) 
- **player3.png** - Spaceship 3 (Character selection option 3)
- **enemy.png** - Enemy spaceship sprite
- **bullet.png** - Laser projectile sprite
- **background.avif** (or .png/.jpg) - Space background image

**Note:** Any image size works! The game automatically scales them.

## Audio/ folder:
Place all sound files here:
- **laser.wav** - Shooting sound effect
- **explosion.wav** - Enemy destruction sound (volume at 10%)
- **game_over.mp3** - Game over sound (volume at 25%)
- **background_music.mp3** (or .ogg/.wav) - Background music loop (volume at 20%)

**Supported audio formats:** .wav, .ogg, .mp3

## Fonts/ folder:
Place custom fonts here:
- **RETRO_SPACE.ttf** - Used for all UI text, scores, menus
- **Oleaguid.ttf** - Used for Game Over screen

## Recommended Sources:
- **Kenney.nl** - Space Shooter Redux pack (images & sounds)
- **OpenGameArt.org** - Free sprites
- **Itch.io** - Asset bundles
- **Freesound.org** - Sound effects
- **DaFont.com** - Free fonts
